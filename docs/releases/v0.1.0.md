# Release v0.1.0

**Date**: November 14, 2025  
**Status**: Initial Release

## Changes

### Initial Implementation

- Frame protocol with STC encryption
- Binary serialization (STT format)
- Varint encoding
- Basic session management
- Basic stream operations
- UDP transport
- WebSocket transport (native RFC 6455)
- Chamber encrypted storage
- Handshake message creation

### Test Coverage

- Overall: 56%
- Frame: 100%
- Serialization: 100%
- Varint: 100%

### Dependencies

- Python 3.9+
- seigr-toolset-crypto >= 0.4.0

### Known Limitations

- Handshake protocol incomplete (only HELLO message)
- Missing stream ordering and flow control
- Transport integration untested
- Session API inconsistencies

### Installation

```bash
pip install -e .
```
